.. admonition:: Oracle Accelerated Data Science (ADS) SDK

  The Oracle Accelerated Data Science (ADS) SDK is a Python library that is included as part
  of the Oracle Cloud Infrastructure Data Science service. ADS offers a friendly user interface,
  with objects and methods that cover all the steps involved in the lifecycle of machine
  learning models, from data acquisition to model evaluation and interpretation.

.. admonition:: Installation

  python3 -m pip install oracle-ads

.. code:: ipython3

  >>> import ads
  >>> ads.hello()
